Project Name: **DIGITAL PRODUCT MARKETPLACE**

Description: 
A MERN‑stack web application developed which allows users to register, list digital products, browse items, and communicate with sellers through an internal messaging system. The system includes a Node.js/Express backend, a React frontend, and a full deployment on AWS EC2 using Nginx and PM2.

Objectives: 
The primary goals of this project include:
1. Build a functional MERN marketplace
Users can list digital products, browse items, and communicate with sellers.
2. Implement secure authentication
JWT‑based login, hashed passwords, protected routes.
3. Develop a messaging system
Users can send and receive messages related to products.
4. Deploy the system on AWS
Using EC2, PM2, Nginx, and GitHub Actions.
5. Demonstrate clean architecture
Separation of concerns, modular code, reusable components.


System Requirements
Functional Requirements: 
1. Users shall be able to register and log in.
2. Users shall be authenticated using JWT.
3. Users shall be able to create, update, and delete product listings.
4. Users shall be able to browse all available products.
5. Users shall be able to send messages to product owners.
6. Users shall be able to view inbox and sent messages.

Non‑Functional Requirements
1. The system shall be deployed on a cloud environment (AWS EC2).
2. The system shall maintain high availability using PM2.
3.The system shall use Nginx as a reverse proxy.
